# Task 2: Excel Basics

## Objective
The objective of this task is to perform data entry and basic data analysis using Microsoft Excel in order to understand essential spreadsheet operations and commonly used functions.

## Task Description
This task involves creating a new Excel workbook and entering a sample dataset such as student grades or sales data. Basic Excel functions are then applied to analyze the data. The task focuses on developing foundational Excel skills required for data handling and analysis.

The following operations are performed:
- Creation of a new Excel workbook
- Entry of a structured sample dataset
- Use of basic Excel functions such as SUM, AVERAGE, and COUNT for analysis

## Tools Used
- Microsoft Excel

## Outcome
Through this task, I developed a basic understanding of spreadsheet operations, data entry techniques, and the application of fundamental Excel functions for data analysis.
